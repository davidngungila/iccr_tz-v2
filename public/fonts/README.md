# Mazzard Font Files

Place your Mazzard font files in this directory.

## Required Font Files:

- `MazzardH-Regular.woff2` (or `.woff`, `.ttf`)
- `MazzardH-Medium.woff2` (or `.woff`, `.ttf`)
- `MazzardH-SemiBold.woff2` (or `.woff`, `.ttf`)
- `MazzardH-Bold.woff2` (or `.woff`, `.ttf`)
- `MazzardH-ExtraBold.woff2` (or `.woff`, `.ttf`)
- `MazzardH-Black.woff2` (or `.woff`, `.ttf`)

## Where to Get Mazzard Fonts:

Mazzard is a premium font family. You can purchase it from:
- MyFonts: https://www.myfonts.com/collections/mazzard-font-pepper-type
- Rentafont: https://rentafont.com/fonts/mazzard-m

## Alternative: Use CDN

If you have Mazzard fonts hosted on a CDN, update the URLs in `resources/views/layouts/app.blade.php` in the `@font-face` declarations.

## Note:

Until the font files are added, the website will use Inter as a fallback font.
