## Features

- Direct upload to Cloudinary from admin dashboard
- Browse and manage all Cloudinary assets
- Search and filter assets
- Delete assets directly from admin
- Select images from Cloudinary when creating/editing content

## Usage

1. Go to Admin Dashboard → Media Library → Cloudinary Assets
2. Browse your Cloudinary media library
3. Upload new assets directly
4. Select images when creating slides or other content
